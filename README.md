## 🤔배경
경력기술서만으로는 제가 어떤식으로 코드를 작성하는지를 보여줄 수 없다고 생각합니다.

그렇기에 만약 React + Typescript로 프로젝트를 진행하게 된다면 어떻게 구성하고 어떻게 개발하는지 보여드리고자 개발하게 되었습니다.

## 👀기술 스택

React, Vite, Typescript, React Query, Context API
